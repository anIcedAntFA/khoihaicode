# Array methods

## Adding and removing elements

The methods `push()`, `pop()`, `unshift()`, and `shift()` allow you to add or remove elements from an array and modify the original array.

- `push()`: Adds an element to the end of the array.
- `pop()`: Removes the last element of the array.
- `unshift()`: Adds an element to the beginning of the array.
- `shift()`: Removes the first element of the array.

## Searching and checking elements

`indexOf()`, `lastIndexOf()`, and `includes()` help find the position or check the existence of an element in an array.

- `indexOf()`: Finds the first index of an element.
- `lastIndexOf()`: Finds the last index of an element.
- `includes()`: Checks if an element exists in the array.

## Converting and reversing the array

`join()`, `toString()`, `and reverse()` allow you to convert an array to a string and reverse the order of the array.

- `join()`: Join the array elements into a string with a separator.
